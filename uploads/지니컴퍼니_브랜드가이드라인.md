# 🧞 지니컴퍼니 브랜드 가이드라인
### Brand Identity Guidelines v1.0

> 이 문서는 지니컴퍼니의 브랜드 정체성을 일관되게 표현하기 위한 공식 가이드라인입니다.  
> 모든 내·외부 커뮤니케이션에 적용됩니다.

---

## 1. 브랜드 철학

### 브랜드 에센스
> **"복잡한 재무를, 당신의 언어로"**

지니컴퍼니는 어렵고 멀게 느껴지는 재무·경영의 세계를  
중소기업과 소상공인 누구나 활용할 수 있도록 **쉽고, 빠르고, 명확하게** 전달합니다.

### 브랜드 퍼스낼리티
지니컴퍼니는 다음 5가지 성격을 가진 브랜드입니다.

| 키워드 | 의미 |
|--------|------|
| **혁신적 (Innovative)** | 관행을 따르지 않고 더 나은 방법을 찾는다 |
| **신뢰할 수 있는 (Trustworthy)** | 숫자와 약속에 책임을 진다 |
| **친근한 (Approachable)** | 전문가답지만 어렵지 않다 |
| **역동적 (Dynamic)** | 빠르게 변하는 시장에 함께 움직인다 |
| **실행하는 (Action-oriented)** | 조언에 그치지 않고 함께 실행한다 |

### 브랜드 포지셔닝
```
          전문성 HIGH
               │
               │       ● 대형 회계법인
               │
               │  ★ 지니컴퍼니
               │    (전문성 + 접근성)
               │
접근성 LOW ────┼──────────────────── 접근성 HIGH
               │
               │            ● 일반 세무사 사무소
               │
          전문성 LOW
```

## 2. 로고

### 로고 컨셉
지니(Genie)의 램프에서 영감을 받아, **소원을 이루어주는 존재**를 상징합니다.  
동시에 숫자 `G`의 순환 구조는 **재무의 흐름과 순환**을 표현합니다.

### 로고 형태

```
┌─────────────────────────────────┐
│                                 │
│   🧞  지니컴퍼니                │
│       GENIE COMPANY             │
│                                 │
└─────────────────────────────────┘
```

### 로고 사용 규칙

**✅ 올바른 사용**
- 배경과 충분한 대비를 유지하여 사용
- 가로형 / 세로형 / 심볼 단독형 중 상황에 맞게 사용
- 최소 사용 크기: 인쇄물 20mm 이상 / 디지털 60px 이상

**❌ 금지 사항**
- 로고 비율 임의 변경
- 로고에 그림자·외곽선·글로우 효과 추가
- 브랜드 컬러 외 색상으로 변경
- 복잡한 패턴·이미지 위에 배치 (가독성 저하 시)
- 로고 회전 또는 기울임

### 여백 규칙 (Exclusion Zone)
로고 주변에는 로고 높이(H)의 **1/2 이상**의 여백을 확보합니다.

```
    ←  H/2  →←──── Logo ────→←  H/2  →
         ↑
        H/2
         ↓
    [  🧞 지니컴퍼니  ]
         ↑
        H/2
         ↓
```

```

|------|--------|------|

| Genie Charcoal | White `#FFFFFF` | 다크 섹션, 푸터 |
| Genie Gold | Genie Charcoal | 경고, 알림 배지 |

**⚠️ 금지 조합**
- Genie Violet + Genie Sky (대비 부족)
- Genie Gold + White (가독성 저하)
- 컬러 위에 동일 계열 컬러 텍스트 사용

### 타이포 스케일

| 등급 | 크기 | 굵기 | 행간 | 용도 |
|------|------|------|------|------|
| Display | 56px / 40pt | ExtraBold | 1.2 | 메인 히어로 제목 |
| H1 | 40px / 28pt | Bold | 1.3 | 섹션 대제목 |
| H2 | 28px / 20pt | SemiBold | 1.4 | 섹션 소제목 |

| Body L | 18px / 13pt | Regular | 1.7 | 주요 본문 |

### 타이포 사용 원칙
- 한 화면에 서체 굵기는 **최대 3종**까지
- 본문 줄 길이는 **1줄 최대 80자 이내** 유지
- 강조는 **Bold 처리 또는 Genie Violet 컬러** 사용 (밑줄 지양)
- 모든 숫자·금액은 **영문 폰트(Plus Jakarta Sans)** 적용

## 5. 아이코노그래피

### 아이콘 스타일
- 스타일: **Line Icon (선형 아이콘)**, 모서리는 Rounded
- 획 두께: **1.5px ~ 2px** (일관성 유지)
- 크기: 16 / 20 / 24 / 32 / 48px (Grid 기준)
- 추천 라이브러리: Lucide Icons / Phosphor Icons

### 아이콘 활용 원칙
- 아이콘 단독 사용 시 반드시 **텍스트 레이블 병기**
- 색상은 **Genie Charcoal** 또는 **Genie Violet** 사용
- 배경 버튼 안에서는 **White** 사용
- 아이콘에 임의로 그라디언트·그림자 적용 금지

### 주요 서비스 아이콘 매핑

| 서비스 | 아이콘 | 의미 |
|--------|--------|------|
| 재무 설계 | 📐 `layout` | 구조 설계 |
| 회계·세무 | 📋 `file-text` | 기록과 신고 |
| 자금조달 | 💡 `zap` | 에너지·연결 |
| 리포트 | 📊 `bar-chart-2` | 데이터 시각화 |
| 상담 | 💬 `message-circle` | 소통 |
| 절세 | 🛡 `shield-check` | 보호 |

## 6. 이미지 & 비주얼 가이드

### 이미지 방향성
지니컴퍼니의 이미지는 **현실적이고 따뜻하되, 전문적인 분위기**를 지향합니다.

**✅ 사용 권장**
- 실제 업무 환경에서의 자연스러운 협업 장면
- 다양한 업종의 소상공인·중소기업 대표 이미지
- 데이터·그래프·태블릿 등 업무 도구가 포함된 장면
- 밝고 자연광이 느껴지는 공간

**❌ 사용 금지**
- 과도하게 연출된 스튜디오 비즈니스 사진 (형식적 악수, 억지 미소)
- 어둡거나 폐쇄적인 공간
- 특정 성별·연령에 편향된 이미지
- 저해상도 이미지 (인쇄: 300dpi 이하 / 디지털: 72dpi 이하)

### 이미지 처리 스타일
- **색감:** 전체적으로 따뜻한 톤, 채도는 살짝 낮게
- **오버레이:** Genie Violet 10~20% 오버레이 허용 (브랜드 통일성)
- **크롭:** 피사체 중심 정렬, 황금 비율(16:9, 4:3, 1:1) 준수

## 7. 보이스 & 톤 (Voice & Tone)

### 브랜드 보이스 원칙

지니컴퍼니는 **스마트하고 친근한 전문가**처럼 말합니다.

```
우리는 이렇게 말합니다            우리는 이렇게 말하지 않습니다
──────────────────────            ──────────────────────────────
명확하고 간결하게                  전문 용어 남발로 어렵게
솔직하고 투명하게                  포장된 말로 모호하게
따뜻하지만 가볍지 않게             딱딱하고 권위적으로
실행 중심의 언어로                 원론적이고 추상적으로
고객 언어로 번역해서               업계 내부 언어 그대로
```

### 채널별 톤 가이드

| 채널 | 톤 | 예시 |
|------|----|----|
| 웹사이트 | 전문적 + 설득적 | "6개월 안에 현금흐름을 바꿉니다." |
| SNS (인스타·블로그) | 친근 + 정보적 | "사장님, 혹시 이런 경험 있으신가요? 👀" |
| 이메일 뉴스레터 | 신뢰 + 유익 | "이번 달 절세 포인트 3가지를 공유드립니다." |
| 제안서·보고서 | 정중 + 데이터 중심 | "분석 결과를 바탕으로 다음을 제안드립니다." |
| 카카오 채널 | 캐주얼 + 빠름 | "안녕하세요! 어떤 부분이 궁금하신가요? 😊" |

### 언어 사용 원칙
- **한 문장 = 한 메시지** 원칙 준수
- 어려운 재무 용어는 반드시 **쉬운 말로 풀어서** 병기
- 숫자와 구체적 사례를 적극 활용 (추상적 표현 지양)
- 고객을 **"사장님"** 또는 **"대표님"**으로 호칭 (존중과 친근함)
- 부정적 표현보다 **긍정·해결 중심** 표현 사용

**예시:**
```
❌ "재무 리스크를 회피하지 못하면 파산할 수 있습니다."
✅ "지금 재무 구조를 점검하면 위기를 기회로 바꿀 수 있습니다."
```

## 8. UI 디자인 원칙

### 레이아웃
- **그리드:** 12 Column Grid (여백 24px, 컬럼 간격 16px)
- **최대 콘텐츠 너비:** 1200px
- **섹션 간격:** 80px ~ 120px (디바이스별 조정)
- **카드 모서리:** `border-radius: 12px`
- **카드 그림자:** `box-shadow: 0 4px 20px rgba(91,63,212,0.08)`

### 버튼 스타일
```
Primary Button
─────────────
배경: Genie Violet (#5B3FD4)
텍스트: White, 16px, SemiBold
패딩: 14px 28px
모서리: 8px
호버: #4A32B0 (10% 어둡게)

Secondary Button
────────────────
배경: Transparent
텍스트: Genie Violet, 16px, SemiBold
테두리: 2px solid #5B3FD4

모서리: 8px

## 9. 명함 & 문서 양식

### 명함 규격
```
크기     : 90mm × 50mm (국내 표준)
전면 구성: 로고 / 이름·직책 / 연락처
후면 구성: 브랜드 슬로건 + Genie Violet 배경

전면 폰트
  이름    : Pretendard Bold, 16pt, Genie Charcoal
  직책    : Pretendard Regular, 10pt, #888888
  연락처  : Plus Jakarta Sans Regular, 9pt, Genie Charcoal

여백     : 상하좌우 6mm 이상
도련     : 3mm
```

### 문서 양식 기본 원칙
- **상단:** 지니컴퍼니 로고 + 문서 제목 (Genie Violet 라인으로 구분)
- **폰트:** 본문 Pretendard Regular 11pt / 제목 Bold 14pt
- **컬러:** 제목 Genie Violet / 본문 Genie Charcoal / 표 헤더 Genie Ivory
- **하단:** 회사명 · 연락처 · 페이지 번호 (회색 소자)

## 10. 브랜드 체크리스트

디자인·콘텐츠 제작 전 아래 항목을 반드시 확인하세요.

### 시각 요소
- [ ] 로고는 지정된 파일(SVG/PNG)을 사용했는가?
- [ ] 브랜드 컬러 코드가 정확히 적용되었는가?
- [ ] 타이포그래피 스케일을 준수했는가?
- [ ] 로고 주변 여백(Exclusion Zone)을 확보했는가?
- [ ] 이미지 해상도가 기준 이상인가?

### 언어·메시지
- [ ] 브랜드 보이스(친근 + 전문)에 맞게 작성되었는가?
- [ ] 재무 전문 용어를 쉽게 풀어 설명했는가?
- [ ] 고객 호칭은 "사장님/대표님"을 사용했는가?
- [ ] 부정적 표현 대신 해결 중심 언어를 사용했는가?

### 디지털 채널
- [ ] 모바일 환경에서 레이아웃이 깨지지 않는가?
- [ ] 버튼 컬러·텍스트가 가이드라인을 따르는가?
- [ ] 접근성(명도 대비 4.5:1 이상)을 충족하는가?
